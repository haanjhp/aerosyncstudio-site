RosterSync 개인정보 처리방침

본 개인정보 처리방침은 2026년 3월 2일부터 적용됩니다.

1. 개인정보의 처리 목적

RosterSync(이하 '앱')는 다음의 목적을 위하여 개인정보를 처리합니다.

  • 앱 회원가입 및 관리: Apple, Google 소셜 로그인을 통한 본인 식별
  • 서비스 제공: 항공사 스케줄 자동 파싱 및 공유 기능 제공

2. 처리하는 개인정보 항목

  • 이메일 주소, 이름, 소속 항공사, UID
  • 스케줄 최소 데이터 (날짜, 비행편명, 출도착지)

※ 중요: 항공사 로그인 정보(ID/비밀번호)는 절대 서버로 전송되지 않으며, 기기 내 Keychain에 암호화 저장됩니다.

3. 개인정보 보유 및 이용 기간

  • 회원 탈퇴 시까지 보유
  • 탈퇴 시 Firebase 데이터 즉시 삭제

4. 제3자 제공

  • 사용자가 설정한 공개 범위 내에서만 스케줄 공유
  • Firebase 클라우드 사용

5. 보안조치

  • HTTPS 암호화 통신
  • Firebase Security Rules 적용
  • Keychain 저장

6. 개인정보 파기

탈퇴 시 모든 데이터는 복구 불가능한 방식으로 삭제됩니다.
